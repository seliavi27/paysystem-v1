# Платежная система (Payment System)

Веб-приложение для управления платежами с поддержкой авторизации пользователей, загрузки аватаров, фильтрации платежей.

##  Описание проекта

Проект представляет собой платежную систему, разработанную в рамках изучения PHP. Приложение позволяет пользователям регистрироваться, авторизовываться, создавать платежи, просматривать историю операций, фильтровать платежи по различным критериям, а также управлять своим профилем (обновлять данные, загружать аватар).

### Основные возможности

-  **Авторизация и регистрация** пользователей
-  **Управление профилем** (изменение имени, телефона, загрузка аватара)
-  **Создание платежей** (различные типы: card, wallet, bank_transfer)
-  **Просмотр истории платежей** с фильтрацией по статусу, валюте, сумме
-  **Сводка по платежам** (количество, общая сумма, средняя, мин, макс)
-  **Смена темы** (светлая/темная) с сохранением в сессии
-  **Загрузка файлов** с валидацией (тип, размер, безопасное имя)
-  **Логирование операций** в файл
-  **Интеграционное тестирование** без браузера

##  Требования

### Системные требования

- **PHP**: 8.1 или выше
- **Веб-сервер**: Apache / Nginx / PHP Built-in Server
- **Расширения PHP**:
  - `json` — для работы с JSON файлами
  - `session` — для управления сессиями
  - `fileinfo` — для определения MIME типов файлов
  - `mbstring` — для работы с многобайтовыми строками

### Проверка версии PHP и расширений

```bash
php -v
php -m | grep -E "json|session|fileinfo|mbstring|openssl"
```
